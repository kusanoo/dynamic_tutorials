# 使い方
サーバを立てる
```
rosrun dynamic_tutorials server.py
```
clientでパラメータを適当に変える
```
rosrun dynamic_tutorials client
```
スライドでパラメータを変える
```
rosrun rqt_reconfigure rqt_reconfigure
```
